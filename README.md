# ChallengeAluraStore

## Descripción
Este repositorio contiene el análisis realizado en Google Colab sobre el desempeño de **cuatro tiendas**. El objetivo es identificar cuál de ellas representa la mejor opción para que el Sr. Juan venda sus productos, considerando métricas de negocio y satisfacción del cliente.

## Factores Analizados
- Ingresos totales por tienda
- Categorías de productos más y menos vendidas
- Calificaciones promedio de los clientes
- Productos más y menos vendidos
- Coste de envío promedio

## Herramientas Utilizadas
- **Google Colab** para el análisis
- **Python (pandas, matplotlib)** para procesamiento y visualización
- **GitHub** para control de versiones y documentación

## Conclusión
El análisis recomienda principalmente la **Tienda 1** como la mejor opción para el Sr. Juan, respaldada por sus ingresos sólidos, alta satisfacción de clientes y costes de envío competitivos.
